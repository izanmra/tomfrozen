BAGAIMANA KAMI BEKERJA?



Simpan file desain 3D dalam format STL

Export 3d model anda dalam format STL atau OBJ untuk dapat diproses oleh printer kami.

Kirim informasi pemesanan melalui E-mail

Kirim file desain ke E-mail tomtomfrozen@gmail.com. Jika perlu bantuan desain silahkan kontak melalui Whatsapp 0877-8860-2037.

Tunggu balasan pemesanan melalui E-mail

Admin kami akan memberikan invoice serta estimasi pengerjaan. Jika dalam 1x24 jam tidak mendapat balasan silahkan Hubungi Kami.

Melakukan pembayaran dan mengirimkan bukti

Lakukan pembayaran dan konfirmasi pembayaran ke Admin kami agar desain anda segera kami proses.

Barang aman hingga ke tempat Anda

Pesanan anda akan kami packing dengan aman agar tiba ditujuan dalam kondisi baik. Jika terjadi keluhan segera Hubungi Kami.
